# StockCategoryAvailability

## Rôle
Regrouper les disponibilités de stock par catégorie.

## Fonction principale
### `groupByCategory(stockAvailabilityMetrics)`
Produit les agrégats de stock visibles dans `BOStatistic`.

## Exemple
```js
const rows = StockCategoryAvailability.groupByCategory(stockAvailabilityMetrics)
```
