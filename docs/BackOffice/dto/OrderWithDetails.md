# OrderWithDetails

## Rôle
Grouper les commandes avec leurs lignes de détail pour faciliter les calculs de ventes et de stock.

## Fonction principale
### `groupOrdersWithDetails(orders, orderDetailsRaw)`
Associe chaque commande à ses détails.

## Exemple
```js
const groups = OrderWithDetails.groupOrdersWithDetails(orders, orderDetailsRaw)
```
