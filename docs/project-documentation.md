## But de la documentation
Centraliser les fiches par page/fonctionnalité pour faciliter la prise en main et la maintenance du projet.

## Comment contribuer
- Pour ajouter/mettre à jour une fiche : créer/éditer `docs/BackOffice/*.md` ou `docs/FrontOffice/*.md`.
- Garder la structure : 1 fiche par composant de page (utilisateur, flux de données, logique métier, relations PrestaShop, dépendances, résumé).
